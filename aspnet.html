Tecnologías para desarrollo de aplicaciones web en ASP.NET

Signal R

Es una biblioteca para los desarrolladores de ASP.NET que simplifica el proceso de agregar funcionalidad web en tiempo real a las aplicaciones. Contiene dos modelos para comunicarse entre clientes y servidores: conexiones persistentes y concentradores.

ASP.NET Core

Es un marco de desarrollo web popular para compilar aplicaciones web en la Plataforma .NET.

Beneficios de ASP.NET CORE

Rendimiento

Es un enfoque clave de ASP.NET Core. Es más rápido que otros marcos web populares en el independiente.

Código abierto

Al igual que el resto de .NET, ASP.NET es de código abierto en GitHub.

Moderno e innovador

Permite que los componentes en ejecución, las API, los compiladores y los lenguajes evolucionen rápidamente, además proporcionan una plataforma estable y compatible para mantener las aplicaciones en ejecución.

Implementación flexible

El entorno de ejecución de ASP.NET Core en el que se ejecuta la aplicación se puede implementar como parte de la aplicación o instalarse de forma centralizada en el servidor web.



Admitido

Puede compilar su propia versión de ASP.NET a partir del código fuente, pero la mayoría de las aplicaciones de producción necesitan una versión compatible de la plataforma.

ASP.NET RAZOR Y BLAZOR
Razor y Blazor

Razor

Sintaxis de marcado que permite incrustar código basado en servidor en páginas web.

Componentes de Razor

  • Cuadrícula web
  • Gráficos web
  • Google Analytics
  • Integración con Facebook
  • Twitter Integración
  • Envío de correo electrónico
  • Validación

Blazor

Marco web de front-end de .NET que admite la representación del lado servidor y la interactividad del cliente en un único modelo de programación.

Componentes de Blazor

Los componentes son clases C# de .NET integradas en ensamblados de .NET que:

  • Definen la lógica de representación de la interfaz de usuario flexible.
  • Controlan acciones del usuario.
  • Se pueden anidar y reutilizar.
  • Se pueden compartir y distribuir como bibliotecas de clases de Razor o paquetes NuGet.
Aplicaciones y Frameworks para desarrollo web con .NET

Aplicaciones Web ASP.NET

Aquellas que siguen el modelo de desarrollo tradicional basado en servidores web, utilizan tecnologías que se generan en el servidor y se envían al cliente.

Aplicaciones Web de una sola página (SPA)

Aquellas que cargan una sola página HTML y actualizan dinámicamente el contenido según la interacción del usuario, sin necesidad de recargar la página completa.

Aplicaciones Web de tiempo real

Aquellas que permiten la comunicación bidireccional entre el cliente y el servidor, lo que permite actualizaciones en tiempo real sin necesidad de que el usuario actualice la página.

Servicios Web, Microservicios y Web APIs

Componentes clave en la arquitectura de aplicaciones web modernas, permiten la comunicación entre diferentes partes de una aplicación o entre aplicaciones diferentes a través de la web, estos servicios suelen seguir estándares como REST o SOAP para definir cómo se comunican.

Servicios Web, Microservicios y Web APIs

Componentes clave en la arquitectura de aplicaciones web modernas, permiten la comunicación entre diferentes partes de una aplicación o entre aplicaciones diferentes a través de la web, estos servicios suelen seguir estándares como REST o SOAP para definir cómo se comunican.

ASP.NET Web Forms

Framework de desarrollo web que forma parte de la plataforma ASP.NET de Microsoft que proporciona un modelo de programación basado en controles y eventos que simplifica el desarrollo de aplicaciones web mediante la abstracción de conceptos como el manejo de eventos del lado del servidor, la gestión del estado de la página y la interacción con los controles de la interfaz de usuario.

ASP.NET MVC (Model-View-Controller)

Framework de desarrollo web que forma parte de la plataforma ASP.NET de Microsoft y que sigue el patrón de diseño Modelo-Vista-Controlador, que separa la lógica de negocio (modelo), la presentación (vista) y el control de flujo (controlador) de una aplicación web, este enfoque proporciona una mayor modularidad y facilita la prueba unitaria de cada componente de la aplicación, lo que hace que el desarrollo sea más estructurado y mantenible.

ASP.NET CORE

Framework de desarrollo web de código abierto y multiplataforma que forma parte de la plataforma ASP.NET de Microsoft, es una versión moderna y modular de ASP.NET, diseñada para funcionar en Windows, macOS y Linux.

Blazor

Framework de desarrollo web de código abierto desarrollado por Microsoft, Blazor permite a los desarrolladores crear aplicaciones web interactivas utilizando C# y .NET en lugar de JavaScript, utiliza WebAssembly para ejecutar código .NET en el navegador, lo que permite a los desarrolladores utilizar el mismo lenguaje y marco de trabajo en el frontend y el backend de sus aplicaciones.

Razor

Framework de Microsoft utilizado principalmente en el desarrollo web con ASP.NET., es una sintaxis de marcado que permite combinar código C# con HTML para crear páginas web dinámicas de forma sencilla y legible.
